## Bump package verions

```sh
  $ npx backstage-cli versions:check --fix
  $ npx backstage-cli versions:bump
```

## Run local
  https://demo.backstage.io/docs/default/component/backstage/getting-started/running-backstage-locally/

```sh
$ kill `lsof -t -i:3000`
$ kill `lsof -t -i:7007`
$ yarn install
$ yarn dev

# Open  url to test
http://localhost:3000
```
